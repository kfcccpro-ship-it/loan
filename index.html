방법서 검색

키워드로 조문 내용을 바로 찾고, 목차에서 편·장을 선택해 빠르게 이동할 수 있습니다.

키워드를 입력하면 관련 조문이 미리보기 카드로 정리됩니다.

목차

편을 펼친 뒤 장을 선택하면 해당 조문만 바로 볼 수 있습니다.

학습용 문제

먼저 범위를 정하고, 기본 세팅인 OX 10문제와 4지선다 10문제로 학습합니다.

기본 세팅
랜덤 범위에서 OX 10문제 → 4지선다 10문제 순서로 제공됩니다. 한 세트를 다 풀면 추가 문제를 이어서 생성합니다.
문제의 범위 설정

전체 범위 랜덤 학습도 가능하고, 특정 편/장/조 구간만 골라 집중 학습도 가능합니다.

현재 세트: OX 10문제 + 4지선다 10문제 0 / 0 완료 정답 0

추가 문제를 진행하시겠습니까?

지금 범위를 유지한 채 새로운 문제 세트를 이어서 풀 수 있습니다.
