# MG인재개발원 여신업무방법서 학습 지원

모바일 중심 학습에 맞춰 아래 방향으로 UI를 재구성한 버전입니다.

- 방법서 검색이 기본 화면
- 키워드 검색 + 목차 선택 중심
- 검색어 노란색 강조
- 학습용 문제는 범위 설정 후 OX 10문제 + 4지선다 10문제 기본 제공
- 현재 기준 업무방법서 버전 표기: 2026.03.12 개정본

주의사항 : 교육용 웹앱으로 업데이트 및 기타세부사항은 직접 엠지니를 활용하시기 바랍니다.


## 추가 반영
- 모바일 터치 영역 확대
- 상단 탭 고정
- 검색 카드 모바일 고정
- 카드/버튼 간격 확대
- 작은 화면 글자 및 줄간격 추가 확대
